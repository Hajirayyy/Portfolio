# Personal CV Website

A modern, responsive CV website built with HTML, CSS, and JavaScript. Features include dark/light mode, downloadable CV, smooth animations, and a working contact form.

## Project Information

This is a professional portfolio website that showcases my skills, education, experience, and projects. The design is clean and modern with smooth animations and works well on all devices.

## Features

### Main Features
- Responsive header with name, title, and photo
- About Me section with key stats
- Education timeline
- Skills organized by category (Frontend, Backend, Tools)
- Experience and Projects sections
- Contact information with social links
- Professional footer

### Extra Features
- **Dark/Light Mode** - Switch between themes with one click, your preference is saved
- **Downloadable CV** - Download my CV with a single click
- **Smooth Animations** - Animations when you scroll, hover effects, and smooth transitions
- **Contact Form** - Working contact form with input validation

## Technologies Used

- **HTML5** - Website structure
- **CSS3** - Styling with modern features like:
  - CSS Variables for easy theme switching
  - Flexbox and Grid for layouts
  - Custom animations
  - Responsive design for all screen sizes
- **JavaScript** - Interactive features:
  - Theme switching
  - Form handling
  - Scroll animations
  - Saving theme preference
- **Google Fonts** - Crimson Pro and DM Sans
- **Git** - Version control
- **Vercel** - Hosting

## Responsive Design

The website adapts to different screen sizes:
- Desktop: screens wider than 968px
- Tablet: 768px to 968px
- Mobile: smaller than 768px
- Small phones: smaller than 480px

## How Features Work

### Dark/Light Mode
- Click the toggle button to switch themes
- Your choice is saved so it stays the same when you come back
- Smooth transition between light and dark

### Downloadable CV
- Click the download button to get a text version of the CV
- Works instantly with one click

### Animations
- Elements fade in as you scroll down
- Buttons and cards have hover effects
- Smooth transitions throughout the site

### Contact Form
- Checks that you've filled in all fields correctly
- Shows success or error messages
- Clears the form after you submit
